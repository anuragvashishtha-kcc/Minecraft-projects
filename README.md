# 🗡️ Skibidi Sword Mod

A simple Minecraft Fabric mod that adds the **Skibidi Sword**, a powerful custom weapon with high attack damage and a special right-click ability.

## ✨ Features

- 🗡️ Custom Skibidi Sword
- 💥 High Attack Damage
- 💪 Right-click to gain **Strength V**
- ⚡ Lightweight and easy to install
- 🎮 Built using the Fabric Mod Loader

---

## 📋 Requirements

- Minecraft **1.21.11**
- Fabric Loader **0.18.1+**
- Fabric API
- Java **21**

---

## 📦 Installation

1. Install the **Fabric Loader** for Minecraft 1.21.11.
2. Download and install the **Fabric API**.
3. Place the mod `.jar` file into your Minecraft `mods` folder.
4. Launch Minecraft using the Fabric profile.
